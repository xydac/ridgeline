# Code of Conduct

## Our Standards

Be respectful. Criticize ideas, not people. Assume good faith. No harassment, personal attacks, or discrimination.

Report concerns to github@mail.xydac.com. Responses are private and prompt.

## Enforcement

Maintainers may remove comments, commits, or contributors who violate these standards.

## Scope

Applies to all project spaces: issues, PRs, discussions, code review, and any representation of the project in public.
